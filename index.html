Vangr

An Arch-Based Haven for Everyday Fólk
Github X Discord

What is Vangr?

Vangr is an early-stage project with four core goals:

  • Deliver powerful task-focused software collections, prioritizing the best tools for the job.
  • Reduce busywork with pre-tuned configs and templates.
  • Build a community rooted in learning and support.
  • Of course, make it all look good too.

Why Vangr?

While Linux has grown easier to use over the years, it still has rough edges. For power users and developers? These edges are easily smoothed. For everyday fólk? Even something simple like gaming has its pain points. I want to change that. Vangr won't be able to smooth every edge, but it can attempt to work down the most obvious ones.

Focused on Power

Other projects already offer a minimalist approach. They're beautiful starting points for a curated garden. Vangr is not that, it's a wild field meant to be sowed and reaped. What does that look like?

Music production can benefit from being greedy. Plugins are the lifeblood of many producers, so the full LADSPA, LV2, and VST plugin meta packages will be included. This is a total of 106 packages, not including dependencies.

For video editing: Kdenlive's optional dependencies are often must-haves for the full toolkit. A proper editor (not just for video) also demands a robust effects library, so Vangr includes any available.

This philosophy means a full Vangr installation will be large, it doesn't mean we're going to go off the deep-end. Effort will be put in to make sure packages aren't included just to satisfy a niche use case.

Communal Improvement

We all have our hot takes and the receipts to back them up. If you think Vangr is missing the mark, I want to hear from you! Join me at the Discord server, Fólkvangr.

Planned Features

  • Task-focused collections provided via a modular installer (e.g., music production, video editing, game dev, office, broadcasting).
  • Beautiful rice (custom desktop/themes), including a Quickshell-powered Wayland shell.
  • Stacking-first or tiling-first window management, powered by Hyprland.
  • Support for both vim and emacs style keyboard bindings
  • Support for keyboard, mouse or hybrid style window management & software operations
  • Hardware auto-detection: Probe your rig and install optimal drivers out of the box.
  • Auto-setup for CachyOS and Chaotic-AUR repos on any Arch base.
  • AI integration for various tools (ex. Krita with ComfyUI)
  • Enhanced Firefox security & performance without reliance on a fork
  • Emphasis on daemonizable software for fast startup speeds, and in cases like terminals, VRAM footprint reduction for multple windows.
  • Performance/gaming mode that disables compositing effects like blur/animations/transparency to lower VRAM footprint
